🍳 PodChef — AI-Powered Kitchen Recommendation Platform

Modular, portable kitchen pods for urban renters — powered by AI

🔗 Live Demo: pod-kitchen-anywhere.lovable.app

🚀 About The Project
PodChef is a live AI-powered web application that helps urban renters find the perfect modular kitchen setup based on their space, budget, and lifestyle.
Built as a full-stack no-code MVP using Lovable and Supabase, the platform has attracted 500+ real registered users since launch.

✨ Features

🤖 AI-Powered Recommendations — Smart kitchen suggestions based on user input
👤 User Authentication — Secure login and registration system
🗄️ Real-time Database — Supabase backend storing 500+ user records
📱 Responsive Design — Works on mobile and desktop
⚡ Fast Deployment — Built and deployed using no-code tools
